### Mathematical Reference

Heat kernel trace on SL(2,Z)\H:
```
K(t) = K_constant(t) + K_discrete(t) + K_continuous(t)

K_constant(t) = 1/(12*t)                                    # vol(SL(2,Z)\H) = pi/3
K_discrete(t) = sum_{j=1}^{N} exp(-(1/4 + r_j^2) * t)      # Maass cusp forms
K_continuous(t) = (1/(4*pi)) * int_0^inf exp(-(1/4+r^2)*t) * psi_scatt(r) dr

psi_scatt(r) = -(phi'/phi)(1/2 + ir)
             = -digamma(ir) + digamma(1/2+ir) - 2*(zeta'/zeta)(2ir) + 2*(zeta'/zeta)(1+2ir)
```

Tail bound for truncation at N terms (Weyl's law):
```
tail <= exp(-lambda_N * t) / (12 * t)    where lambda_N = 1/4 + r_N^2
```
