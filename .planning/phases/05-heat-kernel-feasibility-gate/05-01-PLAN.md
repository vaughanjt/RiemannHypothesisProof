   - `func_mpmath`: Callable taking no args, runs inside `mpmath.workdps(dps + 5)` context. Returns mpmath number.
   - `func_flint`: Callable taking one arg `prec` (bit precision). Returns flint arb or acb.
   - Bit precision: `prec = int(dps * 3.32193) + 20`
   - Extract flint midpoint: `float(fl_val.mid())` if `hasattr(fl_val, 'mid')` else `float(fl_val)`
   - Extract mpmath float: `float(mp_val)`
   - Compute agreement: `-math.log10(abs(fl_mid - mp_float) / abs(mp_float) + 1e-300)` if `mp_float != 0`, else `-math.log10(abs(fl_mid - mp_float) + 1e-300)`
   - `threshold` defaults to `dps - 10`. If `agreement_digits < threshold`, set `flagged=True`.
   - If `agreement_digits < dps - 20`, raise `DualPrecisionError(f"Catastrophic disagreement in {label}: {agreement_digits:.1f} digits")`.
   - Return `DualResult(mpmath_value=mp_val, flint_value=fl_val, agreement_digits=agreement, label=label, flagged=flagged)`.

3. `def dps_to_prec(dps: int) -> int`: Returns `int(dps * 3.32193) + 20`. Utility for converting decimal digits to flint bit precision.
