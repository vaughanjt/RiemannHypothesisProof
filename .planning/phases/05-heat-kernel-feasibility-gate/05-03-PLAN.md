1. `barrier_value_mpmath(L, *, N=None, dps=50) -> mpf`:
   - Re-implement the session41g barrier formula at arbitrary precision using mpmath.
   - `lam_sq = mpmath.exp(mpmath.mpf(str(L)))` (since L = log(lambda^2), so lambda^2 = exp(L))
   - N defaults to `max(15, round(6 * float(L)))` (matching session41g)
   - dim = 2*N+1, ns = range(-N, N+1)
   - w_hat vector: `w_hat[n] = n / (L^2 + (4*pi)^2 * n^2)` for n in ns, w_hat[0]=0, then normalize
   - W02 Rayleigh quotient: `pf = 32 * L * sinh(L/4)^2`, `w_tilde[n] = n / (L^2 + (4*pi)^2 * n^2)`, `w02_rq = -pf * (4*pi)^2 * dot(w_tilde, w_hat)^2`
   - M_prime: sieve primes up to int(float(lam_sq)), for each prime power pk <= lam_sq, compute the matrix contribution. This is the expensive part.
   - M_prime Rayleigh quotient: `mp_rq = dot(w_hat, M_prime @ w_hat)`
   - Return `mpmath.mpf(w02_rq) - mpmath.mpf(mp_rq)` (the barrier B(L) = W02 - M_prime)
   - IMPORTANT: For large lam_sq (>10000), the prime sieve at mpmath precision is slow. Use numpy float64 from session41g as the primary value for the 100-point sweep, and mpmath for spot-checks at key L values. Document this trade-off.
   - Alternative pragmatic approach: wrap `compute_barrier_partial` from session41g (numpy float64, fast) and use it for the bulk comparison. Create `barrier_value_numpy(L, N=None)` that calls session41g's `compute_barrier_partial(exp(L), N)` and returns the float result. Only use mpmath barrier for the highest-precision spot checks (say 10 L values). This avoids reimplementing the entire matrix algebra at arbitrary precision which is not needed for the feasibility gate.

   - **Cross-validation (D-05):** If analytic formula matches numerical fit to 3+ digits at all L values, report the analytic formula as validated. If not, report the discrepancy.
   - Return `{"analytic_candidates": {name: {formula, agreement_digits_by_L}}, "numerical_fits": {L: optimal_t}, "best_candidate": name_or_"implicit", "cross_validation_passed": bool}`
   - Start with a coarse sweep of 10 L values to identify the best candidate, then refine.
