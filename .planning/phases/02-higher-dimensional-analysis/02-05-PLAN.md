    `create_theater_figure(projection_result: ProjectionResult, color_by: np.ndarray | None = None, title: str = "", point_labels: list[str] | None = None) -> go.Figure`:
    - Import plotly.graph_objects as go
    - If target_dim == 3: go.Scatter3d with x, y, z from projection_result.coordinates
    - If target_dim == 2: go.Scatter with x, y
    - color_by: optional array for coloring points (e.g., zero index, spacing, anomaly flag)
    - If color_by provided: set marker.color=color_by, marker.colorscale="Viridis", marker.showscale=True
    - Set marker.size=3 (small points for large datasets), marker.opacity=0.7
    - Layout: title, scene dict with xaxis/yaxis/zaxis labels from projection method + metadata
    - For PCA: label axes "PC1 ({var_explained[0]:.1%})", "PC2 (...)", "PC3 (...)"
    - For t-SNE/UMAP: label axes "Dim 1", "Dim 2", "Dim 3"
    - Add annotation with projection metadata (method, key params)
    - Return go.Figure (user calls fig.show() in Jupyter)

    `create_projection_path_animation(embedding: np.ndarray, methods: list[str] = None, n_frames: int = 20) -> go.Figure`:
    - Default methods: ["pca", "tsne", "umap"]
    - Compute all projections (to 3D)
    - Create animation frames: linearly interpolate coordinates between consecutive projections
    - For interpolation: normalize coordinate ranges, then lerp between method A and method B
    - n_frames: number of interpolation steps between each pair of methods
    - Create go.Figure with frames, animation settings (frame duration, transition)
    - Add play/pause buttons via updatemenus
    - Return figure with animation

    `create_dimension_slice_view(embedding: np.ndarray, fix_dims: dict[int, float], project_remaining: str = "pca") -> go.Figure`:
    - fix_dims: {dim_index: value} -- fix these dimensions to given values
    - Select points where fixed dimensions are within tolerance of target values
    - Tolerance: 0.5 * std of that dimension (so ~68% of points near the value)
    - Project remaining dimensions using specified method
    - Visualize as 2D/3D scatter
    - Add slider or annotation showing which dimensions are fixed and at what value
    - Return figure

    `create_side_by_side(projections: dict[str, ProjectionResult], color_by: np.ndarray | None = None, title: str = "") -> go.Figure`:
    - Create subplot figure with 1 row, N columns (N = number of projections)
    - Each subplot shows the projection as 2D scatter (use first 2 components if 3D)
    - Linked highlighting: hover on point in one subplot highlights same point in all (use customdata + hovertemplate)
    - For Plotly subplots with 3D: use make_subplots with specs=[[{"type": "scene"}, ...]]
    - Return figure
