    `src/riemann/analysis/rmt.py`:

    `generate_gue(n: int, num_matrices: int = 100, seed: int | None = None) -> list[np.ndarray]`:
    - Create RNG: rng = np.random.default_rng(seed)
    - For each matrix: A = (rng.standard_normal((n,n)) + 1j * rng.standard_normal((n,n))) / np.sqrt(2)
    - Hermitianize: H = (A + A.conj().T) / (2 * np.sqrt(n))
    - Compute eigenvalues: np.linalg.eigvalsh(H) (returns sorted real eigenvalues for Hermitian)
    - Return list of eigenvalue arrays

    `generate_goe(n: int, num_matrices: int = 100, seed: int | None = None) -> list[np.ndarray]`:
    - A = rng.standard_normal((n,n))
    - H = (A + A.T) / (2 * np.sqrt(n))
    - Eigenvalues via eigvalsh

    `generate_gse(n: int, num_matrices: int = 100, seed: int | None = None) -> list[np.ndarray]`:
    - Quaternion structure: build 2n x 2n real symplectic matrix
    - A = rng.standard_normal((n,n)) + 1j * rng.standard_normal((n,n))
    - B = rng.standard_normal((n,n)) + 1j * rng.standard_normal((n,n))
    - H_q = [[A, B], [-B.conj(), A.conj()]] (2n x 2n block)
    - H = (H_q + H_q.conj().T) / (2 * np.sqrt(2*n))
    - Eigenvalues come in degenerate pairs; return unique eigenvalues (every other one)

    `_unfold_semicircle(eigenvalues: np.ndarray, n: int) -> np.ndarray`:
    - Wigner semicircle CDF: F(x) = (1/2) + (x*sqrt(4-x^2))/(4*pi) + (1/pi)*arcsin(x/2) for |x| <= 2
    - Scale eigenvalues by sqrt(n) to standard semicircle support [-2, 2]
    - Apply CDF to get unfolded eigenvalues (uniform on [0, n])
    - Private helper, not exported

    `eigenvalue_spacings(eigenvalues_list: list[np.ndarray]) -> np.ndarray`:
    - For each eigenvalue array: unfold using _unfold_semicircle, compute spacings, normalize by mean spacing
    - Concatenate all normalized spacings
    - Return single np.ndarray (same format as spacing.normalized_spacings)

    `wigner_surmise(s: np.ndarray, beta: int = 2) -> np.ndarray`:
    - beta=2 (GUE): (32/pi^2) * s^2 * exp(-4*s^2/pi)
    - beta=1 (GOE): (pi/2) * s * exp(-pi*s^2/4)
    - beta=4 (GSE): (2^18 / (3^6 * pi^3)) * s^4 * exp(-64*s^2/(9*pi))
    - Raise ValueError for other beta values

    `fit_effective_n(target_spacings: np.ndarray, n_range: tuple[int, int] = (10, 500), num_matrices: int = 200, seed: int = 42) -> dict`:
    - For N in range(n_range[0], n_range[1], step=10): generate GUE(N), compute spacings, compute chi-squared against target_spacings histogram
    - Return {"best_n": int, "chi_squared": float, "all_fits": dict[int, float]}
    - This is the residual analysis tool the user wants ("at what effective N do zeros best match GUE?")
