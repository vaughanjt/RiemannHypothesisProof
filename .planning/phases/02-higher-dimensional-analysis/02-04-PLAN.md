    `@dataclass` `Anomaly`:
    - zero_range: tuple[int, int]  -- (start_index, end_index) of anomalous window
    - statistic: str  -- which statistic deviated (e.g., "mean_spacing", "spacing_variance", "pair_correlation")

    `detect_anomalies(zeros: list[ZetaZero], window_size: int = 50, stride: int = 25, sigma_thresholds: dict[str, float] | None = None) -> list[Anomaly]`:
    - Default thresholds: {"info": 2.0, "warning": 3.0, "critical": 4.0}
    - Slide window over zeros with given stride
    - For each window, compute:
      1. Mean spacing deviation: mean of normalized_spacings vs expected 1.0
         - Standard error: sqrt(GUE_VARIANCE / window_size) where GUE_VARIANCE = 1 - 2/pi^2 approx 0.2732
      2. Spacing variance deviation: variance of normalized spacings vs GUE expected variance
         - Use chi-squared-based standard error for variance
      3. Pair correlation deviation at x=1.0: compare observed r_2(1.0) against GUE prediction
    - For each statistic, compute z-score = |observed - expected| / standard_error
    - Assign severity: iterate thresholds from highest to lowest, first threshold exceeded determines severity
    - Create Anomaly for any z_score > min(sigma_thresholds.values())
    - Return sorted by sigma_deviation (most anomalous first)

    GUE reference constants (hardcoded, from theory):
    - GUE_MEAN_SPACING = 1.0
    - GUE_SPACING_VARIANCE = 1.0 - 2.0 / (np.pi ** 2)  # approximately 0.2732
    - GUE_PAIR_CORRELATION_AT_1 = 1.0 - (np.sin(np.pi) / (np.pi)) ** 2  # = 1.0 (since sin(pi)=0)
      Wait -- gue_pair_correlation(1.0) = 1 - (sin(pi)/pi)^2 = 1 - 0 = 1.0. That seems wrong.
      Actually: r_2(x) = 1 - (sin(pi*x)/(pi*x))^2. At x=1: 1 - (sin(pi)/pi)^2 = 1 - 0 = 1.0
      This is correct: at exactly one mean spacing, the pair correlation is 1 (full correlation).
      Better to use x=0.5 or a range. For anomaly detection, compare the full pair correlation curve.
      Simpler approach: just check mean and variance of spacings. Pair correlation check can use the
      integrated difference: sum |r_2_observed(x) - r_2_gue(x)| over x range.
