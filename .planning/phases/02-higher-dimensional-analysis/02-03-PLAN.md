Implement the embedding coordinates (feature extractors), HDF5 storage, and projection pipeline -- the heart of the higher-dimensional analysis framework.

Purpose: This plan connects zero data to visual exploration. Feature extractors turn ZetaZero objects into N-dimensional points. Projections compress those points to 2D/3D for visualization. HDF5 stores the results for fast retrieval. Together they enable the "cast the widest net" approach the user wants.
Output: Working embedding pipeline (extract -> store -> project) with 5+ feature extractors and 4 projection methods.

    `src/riemann/embedding/coordinates.py`:

    All extractors have signature: `(zeros: list[ZetaZero], *, dps: int = 50) -> np.ndarray`
    All return float64 ndarray of length len(zeros).

    Implement these extractors (fast, no precision-critical computation):

    `extract_imaginary_part(zeros, *, dps=50) -> np.ndarray`:
    - Return np.array([float(z.value.imag) for z in zeros], dtype=np.float64)

    `extract_left_spacing(zeros, *, dps=50) -> np.ndarray`:
    - Sort zeros by imaginary part
    - Compute diff of imaginary parts
    - Pad with 0.0 at start (first zero has no left neighbor)
    - Normalize by local mean spacing: 2*pi/log(t/(2*pi))

    `extract_right_spacing(zeros, *, dps=50) -> np.ndarray`:
    - Same as left but pad with 0.0 at end

    `extract_zeta_derivative_magnitude(zeros, *, dps=50) -> np.ndarray`:
    - For each zero, compute |zeta'(rho)| using mpmath.zeta(zero.value, derivative=1) inside precision_scope(dps)
    - Convert magnitude to float64
    - CACHE results: check if a cached file exists at DATA_DIR/cache/zeta_derivatives_{zero_range_hash}.npy
    - If cache hit, load and return. If cache miss, compute, save, return.
    - This is the expensive feature -- log a warning if computing more than 100 zeros

    `extract_local_density_deviation(zeros, *, dps=50) -> np.ndarray`:
    - Expected local density at height t: log(t/(2*pi)) / (2*pi)
    - Actual local density: count zeros in window of size w around each zero / w
    - Window size w = 10 * (2*pi / log(t/(2*pi))) -- 10 mean spacings
    - Return (actual - expected) / expected for each zero

    `extract_pair_correlation_local(zeros, *, dps=50) -> np.ndarray`:
    - For each zero, compute pair correlation r_2(1.0) using a local window of 50 surrounding zeros
    - Return the local r_2 value at x=1.0 for each zero

    `extract_hardy_z_sign_changes(zeros, *, dps=50) -> np.ndarray`:
    - For each zero, count Hardy Z function sign changes in a local neighborhood
    - Use engine/lfunctions.py if hardy_z is available, otherwise use mpmath.siegelz
    - Return count of sign changes in window of 10 mean spacings around each zero

    `extract_local_entropy(zeros, *, dps=50) -> np.ndarray`:
    - For each zero, compute Shannon entropy of spacings in a local window of 50 zeros
    - Use binned method with 10 bins (fast approximation)

    `extract_compression_distance(zeros, *, dps=50) -> np.ndarray`:
    - For each zero, compute Lempel-Ziv complexity of binarized spacings in local window
    - Normalize by theoretical maximum complexity for window size

    `compute_embedding(config: EmbeddingConfig, zeros: list[ZetaZero]) -> np.ndarray`:
    - For each feature_name in config.feature_names: look up FEATURE_EXTRACTORS[feature_name], call it
    - Stack columns: np.column_stack(features)
    - Apply scaling per config.scaling:
      - "standard": sklearn.preprocessing.StandardScaler().fit_transform(embedding)
      - "robust": sklearn.preprocessing.RobustScaler().fit_transform(embedding)
      - "none": return raw
    - Return ndarray of shape (len(zeros), len(config.feature_names))

    After defining all extractors, UPDATE FEATURE_EXTRACTORS in registry.py:
    - Import coordinates module and replace stubs with real functions
    - Keep registry.py's FEATURE_EXTRACTORS dict as the single source of truth
    - In coordinates.py, define all extractor functions, then at module level:
      `from riemann.embedding.registry import FEATURE_EXTRACTORS`
      `FEATURE_EXTRACTORS.update({...})` with real implementations

    IMPORTANT: To avoid circular imports, use a registration pattern:
    - registry.py defines FEATURE_EXTRACTORS with stubs
    - coordinates.py imports FEATURE_EXTRACTORS and replaces stubs on import
    - __init__.py imports coordinates AFTER registry to trigger registration

    `src/riemann/viz/projection.py`:

    `@dataclass` `ProjectionResult`:
    - coordinates: np.ndarray  (n_points, target_dim)
    - method: str
    - source_dim: int
    - target_dim: int
    - metadata: dict

    `project_pca(data: np.ndarray, n_components: int = 3) -> ProjectionResult`:
    - from sklearn.decomposition import PCA
    - pca = PCA(n_components=n_components)
    - coords = pca.fit_transform(data)
    - metadata: variance_explained (list), total_variance_explained (float), components (list of lists)
    - Clamp n_components to min(n_components, data.shape[1], data.shape[0])

    `project_tsne(data: np.ndarray, n_components: int = 3, perplexity: float = 30.0, random_state: int = 42) -> ProjectionResult`:
    - from sklearn.manifold import TSNE
    - Auto-reduce perplexity if n_samples < 3 * perplexity (sklearn requirement)
    - If data.shape[1] > 50: pre-reduce with PCA(50) per sklearn docs recommendation
    - tsne = TSNE(n_components=n_components, perplexity=perplexity, random_state=random_state)
    - coords = tsne.fit_transform(data)
    - metadata: perplexity, kl_divergence, pre_reduced (bool)

    `project_umap(data: np.ndarray, n_components: int = 3, n_neighbors: int = 15, min_dist: float = 0.1, random_state: int = 42) -> ProjectionResult`:
    - import umap
    - Auto-reduce n_neighbors if n_samples < n_neighbors
    - reducer = umap.UMAP(n_components=n_components, n_neighbors=n_neighbors, min_dist=min_dist, random_state=random_state)
    - coords = reducer.fit_transform(data)
    - metadata: n_neighbors, min_dist

    `project_stereographic(data: np.ndarray) -> ProjectionResult`:
    - Normalize to unit sphere: data / ||data|| per row
    - Stereographic from north pole: x_i / (1 - x_n) for i < n
    - Clip denominator to avoid division by zero: np.clip(1 - x_n, 1e-10, None)
    - metadata: projection_pole = "north"
