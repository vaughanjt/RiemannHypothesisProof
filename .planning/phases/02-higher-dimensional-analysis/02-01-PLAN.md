    Step 4: Write GREEN implementation in `src/riemann/analysis/spacing.py`:

    `normalized_spacings(zeros: list[ZetaZero]) -> np.ndarray`:
    - Extract imaginary parts as float64 array, sort
    - Compute raw spacings via np.diff
    - Compute local mean spacing at midpoints: 2*pi / log(t/(2*pi))
    - Return raw_spacings / mean_spacings
    - Raise ValueError if len(zeros) < 2

    `pair_correlation(spacings: np.ndarray, bins: int = 200, x_range: tuple[float, float] = (0.0, 4.0)) -> tuple[np.ndarray, np.ndarray]`:
    - Compute all pairwise normalized gaps (not just nearest-neighbor)
    - For N spacings, accumulate gaps s_i + s_{i+1} + ... + s_{i+k} for k=0,1,2,...
    - Histogram into bins over x_range
    - Normalize to get density estimate
    - Return (x_centers, r2_values)

    `gue_pair_correlation(x: np.ndarray) -> np.ndarray`:
    - Return 1 - (sin(pi*x)/(pi*x))^2
    - Handle x=0 case: result[x==0] = 0.0 (since sinc(0)=1, 1-1=0... wait, sin(0)/0 = 1 via L'Hopital, so 1-1^2=0)
    - Actually: gue_pair_correlation(0) = 1 - 1 = 0. At x -> inf, approaches 1.

    `n_level_density(spacings: np.ndarray, n: int = 2, bins: int = 100) -> np.ndarray`:
    - For n=2: compute density of consecutive n-tuples of normalized spacings
    - Sum of n consecutive spacings, histogram
    - Return density array

    `number_variance(spacings: np.ndarray, L_values: np.ndarray | None = None) -> np.ndarray`:
    - Sigma_2(L) = variance of the number of spacings in intervals of length L
    - For each L in L_values, slide a window of length L over cumulative spacings, count zeros in window, compute variance
    - Default L_values: np.linspace(0.1, 5.0, 50)

    All functions: function-based API (no classes), return numpy arrays, never plot.
