```python
@dataclass
class BostConnesResult:
    beta: float
    partition_value: float
    kms_values: np.ndarray
    zeta_comparison: float  # |Z(beta) - zeta(beta)|
    metadata: dict
```

2. `bost_connes_partition(beta: float, n_max: int = 1000) -> float`:
   - Z(beta) = sum_{n=1}^{n_max} n^{-beta}
   - For beta > 1 this converges to zeta(beta)
   - For beta <= 1 the sum diverges; raise ValueError with message explaining beta > 1 is required for convergence
   - Return float value
   - Use numpy vectorized: np.sum(np.arange(1, n_max+1, dtype=float) ** (-beta))

3. `bost_connes_kms_values(beta: float, n_max: int = 50) -> np.ndarray`:
   - KMS state: phi_beta(e_n) = n^{-beta} / Z(beta)
   - Compute Z = bost_connes_partition(beta, n_max)
   - Return array of n^{-beta}/Z for n = 1, ..., n_max
   - These sum to 1.0 (probability distribution)

4. `phase_transition_scan(beta_range: tuple[float, float] = (1.01, 5.0), n_points: int = 50, n_max: int = 500) -> dict`:
   - Scan beta values from beta_range[0] to beta_range[1]
   - At each beta: compute partition function, KMS values, Shannon entropy of KMS distribution
   - Entropy = -sum(p * log(p)) for p = KMS values (with p*log(p) = 0 when p = 0)
   - Return {"betas": ndarray, "partition_values": ndarray, "kms_entropy": ndarray, "d_entropy_d_beta": ndarray}
   - The derivative d_entropy/d_beta shows the phase transition: sharp change near beta=1

5. `compute_bost_connes(beta: float, n_max: int = 500) -> BostConnesResult`:
   - Convenience function combining partition function, KMS values, and comparison to mpmath.zeta
   - zeta_comparison = abs(Z(beta) - float(mpmath.zeta(beta)))
   - Return BostConnesResult

```python
class AnalogyMapping:
```

2. `create_analogy_mapping(source_domain: str, target_domain: str, correspondences: dict[str, str], unknowns: list[str] | None = None, confidence: float = 0.0) -> AnalogyMapping`:
   - Convenience constructor with defaults
   - unknowns defaults to empty list
   - evidence starts empty

3. `test_correspondence(mapping: AnalogyMapping, source_data: np.ndarray, target_data: np.ndarray, metric: str = "ks") -> dict`:
   - Compare source_data distribution against target_data distribution
   - If metric == "ks": scipy.stats.ks_2samp for Kolmogorov-Smirnov
   - If metric == "chi_squared": histogram-based chi-squared (same approach as spectral_comparison)
   - If metric == "correlation": Pearson correlation of sorted, length-matched arrays
   - Return {"metric": str, "statistic": float, "pvalue": float, "ks_pvalue": float (if KS), "chi_squared": float (if chi2), "correlation": float (if corr), "n_source": int, "n_target": int}

   - Return experiment_id

5. `load_analogy_from_workbench(experiment_id: str, db_path: str | Path | None = None) -> AnalogyMapping | None`:
   - Load experiment via load_experiment
   - Reconstruct AnalogyMapping from experiment["parameters"] via from_dict()
   - Return None if experiment not found
