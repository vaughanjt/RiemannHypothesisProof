```python
class ModularFormResult:
    weight: int

    coefficients: list[complex]

```

2. `eisenstein_series(k: int, n_terms: int = 50, dps: int = 15) -> list[float]`:
   - Compute normalized Eisenstein series E_k(q) = 1 + (2k/B_k) * sum_{n=1}^{n_terms} sigma_{k-1}(n) * q^n
   - B_k = Bernoulli number via mpmath.bernoulli(k)
   - sigma_{k-1}(n) = sum of (k-1)th powers of divisors of n
   - Return list of Fourier coefficients [a_0, a_1, ..., a_{n_terms-1}]
   - For k=4: normalization gives E_4 = 1 + 240*q + 2160*q^2 + ...
   - For k=6: E_6 = 1 - 504*q - 16632*q^2 + ...
   - Only even k >= 4 are valid; raise ValueError for odd or k < 4

3. `_divisor_sigma(n: int, k: int) -> int`:
   - Helper: sum of k-th powers of divisors of n
   - sigma_k(n) = sum(d^k for d in range(1, n+1) if n % d == 0)

4. `compute_q_expansion(weight: int, level: int = 1, n_terms: int = 50, dps: int = 15) -> ModularFormResult`:
   - For level=1: construct cusp forms from Eisenstein series
   - The unique cusp form of weight 12, level 1 is Delta = (E_4^3 - E_6^2) / 1728
   - For weight 12: compute via Delta formula above
   - For other weights at level 1: if no cusp form exists (weight < 12), return the Eisenstein series
   - Return ModularFormResult with computed coefficients

5. `hecke_eigenvalues(weight: int, level: int = 1, primes: list[int] | None = None, n_terms: int = 100) -> dict[int, complex]`:
   - For cusp forms that are eigenforms (like Delta), the Fourier coefficients ARE the Hecke eigenvalues: T_p(f) = a_p * f
   - Compute q-expansion to n_terms, extract a_p for each prime p in the list
   - Default primes: [2, 3, 5, 7, 11, 13]
   - Return dict mapping prime -> eigenvalue
   - Known values for Delta: tau(2)=-24, tau(3)=252, tau(5)=-4830, tau(7)=16744

2. `LMFDBError(Exception)` custom exception

3. Cache schema (SQLite):
```sql
CREATE TABLE IF NOT EXISTS lmfdb_cache (
    query_key TEXT PRIMARY KEY,
    response_json TEXT NOT NULL,
    collection TEXT NOT NULL,
    cached_at TEXT NOT NULL
);
```

4. `_init_cache(cache_db: str | Path) -> str` -- idempotent cache table creation

5. `_cache_key(collection: str, params: dict, fields: list[str] | None) -> str`:
   - SHA-256 of collection + sorted params JSON + sorted fields
   - Deterministic key for cache lookup

6. `query_lmfdb(collection: str, params: dict, fields: list[str] | None = None, cache_db: str | Path | None = None, refresh: bool = False) -> list[dict]`:
   - Build URL: `{LMFDB_BASE}/{collection}/?{params}&_format=json`
   - If fields: add `_fields={','.join(fields)}`
   - Check cache first (unless refresh=True)
   - On cache miss: `requests.get(url, timeout=30)`
   - Handle pagination: if response has `next` field, follow up to 10 pages
   - On HTTP error: raise LMFDBError with status code and URL
   - Cache response JSON in SQLite
   - Return list of result dicts (from response["data"])

7. Convenience functions:
   - `get_lfunction(label: str, cache_db=None) -> dict | None`
   - `get_modular_form(label: str, cache_db=None) -> dict | None`
   - `get_number_field(label: str, cache_db=None) -> dict | None`

8. `clear_cache(cache_db: str | Path | None = None) -> int`:
   - Delete all rows from lmfdb_cache table
   - Return number of entries deleted
