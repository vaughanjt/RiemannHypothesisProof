2. `construct_berry_keating_box(n: int, L: float = 10.0) -> np.ndarray`:
   - Grid points x = linspace(L/n, L, n) avoiding x=0 singularity
   - Symmetrized H = (xp + px)/2 discretized via central finite differences
   - H[i,i-1] = -x[i]/(2*dx), H[i,i+1] = x[i]/(2*dx), then H_sym = (H + H.T)/2
   - Returns real symmetric (n, n) matrix

3. `construct_berry_keating_smooth(n: int, L: float = 10.0, V_strength: float = 0.1) -> np.ndarray`:
   - Same as box but adds smooth confining potential V(x) = V_strength * x^2 along diagonal
   - H_smooth = H_sym + diag(V_strength * x^2)
   - Returns real symmetric (n, n) matrix

4. `compute_spectrum(matrix: np.ndarray, operator_name: str = "unknown") -> SpectralResult`:
   - Uses scipy.linalg.eigh for dense, scipy.sparse.linalg.eigsh for sparse
   - Sorts eigenvalues ascending
   - Returns SpectralResult with chi_squared_fit=0.0 (not yet compared), empty metadata

5. `spectral_comparison(eigenvalues: np.ndarray, zero_heights: np.ndarray) -> dict`:
   - Normalize both spacing distributions to mean 1
   - Compute chi-squared on 40-bin histograms over [0, 4]
   - Compute KS test via scipy.stats.ks_2samp
   - Return {"chi_squared": float, "ks_statistic": float, "ks_pvalue": float, "n_eigenvalues": int, "n_zeros": int}

```python
@dataclass
class TraceFormulaResult:
    x: float

    relative_error: float
    metadata: dict
```

2. `chebyshev_psi_exact(x: float) -> float`:
   - Compute psi(x) = sum of Lambda(n) for n <= x, where Lambda(n) = log(p) if n = p^k, else 0
   - Use sympy.isprime or trial division to identify prime powers
   - For x up to ~10000 this is fast enough via direct enumeration
   - Return float value

3. `weil_explicit_psi(x: float, zeros: list[float], n_terms: int | None = None) -> float`:
   - Implements: psi(x) approx x - sum_{rho} (x^rho / rho) - log(2*pi) - 0.5*log(1 - x^{-2})
   - zeros are positive imaginary parts of non-trivial zeros (e.g., [14.134725..., 21.022039..., ...])
   - Each zero t contributes a conjugate pair: rho = 0.5+it, rho_conj = 0.5-it
   - term = x^rho / rho + x^(rho_conj) / rho_conj (take real part)
   - Truncate at n_terms zero pairs (default: all)
   - Return float approximation

4. `explicit_formula_terms(x: float, zeros: list[float], max_terms: int = 50) -> list[tuple[int, float]]`:
   - Compute weil_explicit_psi for n_terms = 1, 2, 4, 8, 16, ..., max_terms (powers of 2 up to max_terms, plus max_terms itself)
   - Return list of (n_terms, psi_approx) pairs showing convergence
   - Useful for visualizing how adding more zeros improves the approximation

5. `compute_trace_formula(x: float, zeros: list[float], n_terms: int | None = None) -> TraceFormulaResult`:
   - Convenience function combining exact and approximate
   - Returns TraceFormulaResult with both values and relative error
